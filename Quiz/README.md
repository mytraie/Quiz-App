# Quiz app

**Description:**
This quiz app is designed for assessing users through a timer-based quiz system. The app follows the MVVM architecture and incorporates a scoring system where participants can earn points for correct answers, gain extra points for answering within 20 seconds, and face penalties for incorrect answers.

## Features
- Multiple-choice questions
- Timer for each question
- Scoring system
- User-friendly interface

## Architecture
The app follows the MVVM (Model-View-ViewModel) architecture, separating concerns to enhance maintainability and scalability.

## Usage
- Launch the app.
- Answer questions within the timer limit.
- Earn points for correct answers.
- Gain extra points for answering within 20 seconds.
- Receive penalties for incorrect answers.

## Scoring System
- Correct Answer: +4 points
- Answered within 20 seconds: +1 point (bonus)
- Wrong Answer: -2 points
